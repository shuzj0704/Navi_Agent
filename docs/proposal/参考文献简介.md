## 1. 导航基础架构与双系统

> 双系统架构（System 2 慢规划 + System 1 快执行）、联合训练、异步推理。


### Ground Slow, Move Fast: A Dual-System Foundation Model for Generalizable Vision-and-Language Navigation

- **作者**: Meng Wei, Chenyang Wan, Jiaqi Peng 等 | 通讯: **Xihui Liu**（The University of Hong Kong）
- **发表**: ICLR 2026 submission
- **输入**: 多视角 RGB 图像 + 自然语言导航指令
- **输出**: System 2 输出 pixel goal (u,v) + latent goal embedding；System 1 输出连续无碰撞轨迹
- **功能与效果**: 首个双系统 VLN 基础模型。System 2（7B VLM, 2Hz）做高层规划，System 1（轻量 DiT, 30Hz）做低层执行。VLN-CE 和 VLN-PE 上 SOTA，支持真实机器人部署
- **创新点**: (1) 将 Kahneman 双系统认知理论引入 VLN，解耦规划和执行频率；(2) System 2 同时输出 pixel goal（可解释）和 latent goal（语义丰富）条件化 System 1；(3) 解耦训练保留 VLM 泛化能力
- **与 NaviAgent 的关联**: NaviAgent 直接继承其双系统架构，System 2 (2Hz) + System 1 DiT (30Hz) 的频率设计和 dual-head 输出方案均源于此



### InternVLA-N1: An Open Dual-System Vision-Language Navigation Foundation Model

- **作者**: Meng Wei, Chenyang Wan, Jiaqi Peng 等 | 通讯: **Xihui Liu**（HKU / Shanghai AI Lab）
- **发表**: arXiv, 2025
- **输入**: 历史 egocentric RGB 图像序列 + 语言指令（System 2）；RGB + Depth + goal embedding（System 1）
- **输出**: System 2 输出 pixel goal / latent plan tokens；System 1 输出 30Hz 控制轨迹
- **功能与效果**: 开源双系统 VLN 基础模型。在 53M 图像、800K 指令、3154 场景上训练。System 1 支持 point-goal / image-goal / no-goal 三种导航模式，零样本泛化到轮式/足式/人形机器人
- **创新点**: (1) InternData-N1 大规模自动数据生成管线（A* 路径 + ESDF 优化 + VLM caption）；(2) System 1 的统一 goal-conditioned 策略；(3) joint training 的 latent plan tokens
- **与 NaviAgent 的关联**: NaviAgent 的 SFT 训练流程（数据混合策略、LR 调度、ZeRO-2 配置）主要参考此工作。pixel goal 标注方式也沿用其方案



### GR00T N1: An Open Foundation Model for Generalist Humanoid Robots

- **作者**: Johan Bjorck, Fernando Castaneda 等 40+ | 通讯: **Yuke Zhu**（NVIDIA Research / UT Austin）
- **发表**: arXiv 2503.14734
- **输入**: RGB 图像 + 语言指令 + 本体感知（关节角度等）
- **输出**: 连续关节控制信号
- **功能与效果**: 面向人形机器人的通用双系统架构。VLM + DiT Action Expert 端到端联合训练，混合真实轨迹、人类视频和 NVIDIA Omniverse 合成数据
- **创新点**: (1) 双系统联合端到端训练（非分离训练），梯度从 Action Expert 回传到 VLM；(2) 大规模异构数据混训（real + human video + sim）
- **与 NaviAgent 的关联**: NaviAgent 的 System 1 + System 2 联合训练方案参考了其梯度回传设计



### RationalVLA: A Rational Vision-Language-Action Model with Dual System

- **作者**: Wenxuan Song, Jiayi Chen 等 | 通讯: **Haoang Li**（HKUST(GZ)）
- **发表**: arXiv 2506.10826
- **输入**: RGB 图像 + 语言指令
- **输出**: `<ACT>` token（接受执行）或 `<REJ>` token（拒绝执行）。`<ACT>` 位置的 hidden state → latent goal → 扩散策略生成动作
- **功能与效果**: VLM 学会自主判断指令是否可执行，通过 `<ACT>`/`<REJ>` 实现选择性执行
- **创新点**: (1) `<ACT>` token 的 hidden state 直接作为扩散策略的 latent goal（零信息损失）；(2) 可学习的 latent interface 实现 VLM + DiT 端到端训练
- **与 NaviAgent 的关联**: **NaviAgent 的 `<act>` token 设计直接参考此工作**——hidden state 同时送入 Spatial Head 和 Latent Goal Head



### TIC-VLA: Think-in-Control VLA for Robot Navigation in Dynamic Environments

- **作者**: Zhiyu Huang, Yun Zhang 等 | 通讯: **Jiaqi Ma**（UCLA）
- **发表**: arXiv 2602.02459
- **输入**: RGB 图像 + 语言指令 + 延迟元数据（标记 VLM 推理完成时间）
- **输出**: 低延迟导航动作（补偿 VLM 推理延迟）
- **功能与效果**: 显式建模 VLM 推理延迟与反应控制之间的异步问题。Action Expert 条件化于 cached VLM hidden states + latency metadata。3-stage latency-consistent training
- **创新点**: (1) 将推理延迟视为一等设计参数（而非最小化目标）；(2) 训练时注入真实推理延迟
- **与 NaviAgent 的关联**: System 2 (2Hz) / System 1 (30Hz) 异步通信的理论参考



### HiRT: Enhancing Robotic Control with Hierarchical Robot Transformers

- **作者**: Jianke Zhang, Yanjiang Guo 等 | 通讯: **Jianyu Chen**（Tsinghua University / Shanghai Qi Zhi）
- **发表**: CoRL 2025
- **输入**: RGB 图像 + 语言指令（VLM 低频）；VLM latent + 实时 RGB（Policy 高频）
- **输出**: 机器人操作动作
- **功能与效果**: VLM 低频运行捕获时间不变特征，轻量 policy 在 VLM latent conditioning 下高频运行。推理延迟减少 58%，动态操作 48% → 75%
- **创新点**: (1) 异步双频架构（VLM + Policy 解耦频率）；(2) VLM output 作为 latent conditioning signal
- **与 NaviAgent 的关联**: 验证了 2Hz/30Hz 频率分离设计的合理性



### DiffusionVLA: Scaling Robot Foundation Models via Unified Diffusion and Autoregression

- **作者**: Junjie Wen, Yichen Zhu 等 | 通讯: **Yichen Zhu / Feifei Feng**（Midea Group / 华东师大）
- **发表**: ICML 2025
- **输入**: RGB 图像 + 语言指令
- **输出**: 自回归推理 token + 扩散生成的连续动作
- **功能与效果**: 统一自回归推理 + 扩散动作生成。Reasoning injection module 将推理直接集成到策略学习。2B 模型在单 A6000 上 82Hz
- **创新点**: (1) 无缝融合 autoregressive reasoning + diffusion action generation；(2) 从 2B 到 72B 的可扩展设计
- **与 NaviAgent 的关联**: System 2（autoregressive）+ System 1（diffusion）组合的性能预期参考

## 2. 室内导航：VLN 指令跟随与推理

> 给定 turn-by-turn 指令跟随导航。含 CoT 推理、强化微调、记忆增强、拓扑图方法。


### NavCoT: Boosting LLM-Based Vision-and-Language Navigation via Learning Disentangled Reasoning

- **作者**: Bingqian Lin, Yunshuang Nie 等 | 通讯: **Xiaodan Liang**（中山大学）
- **发表**: AAAI 2024
- **输入**: 全景 RGB 图像 + 导航指令 + 候选方向
- **输出**: 推理链文本（CoT）+ 选择的导航方向
- **功能与效果**: 将 CoT 推理引入 VLN，模型在选择动作前先输出推理文本。R2R/REVERIE/CVDN 上一致提升
- **创新点**: (1) 首次在 VLN 中引入 structured CoT；(2) 设计了导航特定的推理模板（scene description → instruction alignment → action decision）
- **与 NaviAgent 的关联**: NaviAgent 认知推理的早期启发来源，其推理模板影响了 thinking text 的标注设计



### Aux-Think: Exploring Reasoning Strategies for Data-Efficient Vision-Language Navigation

- **作者**: Shuo Wang, Yongcai Wang 等 | 通讯: **Zhaoxin Fan**（中国人民大学 / 快手）
- **发表**: arXiv 2505.11886
- **输入**: 全景 RGB + 导航指令
- **输出**: 对比三种策略——No-Think（直接动作）/ Pre-Think（先推理再动作）/ Post-Think（先动作再推理）
- **功能与效果**: 发现 **Inference-time Reasoning Collapse**：推理时输出 CoT 反而降低 VLN 精度。提出 Aux-Think 策略：训练时用 CoT 做辅助 loss，推理时不输出 CoT。发布 R2R-CoT-320k 数据集
- **创新点**: (1) 首次系统评估 VLN 推理策略并发现 reasoning collapse；(2) Aux-Think 策略（训练时 CoT 辅助，推理时跳过）；(3) 大规模 VLN CoT 数据集
- **与 NaviAgent 的关联**: **直接影响核心设计决策**——NaviAgent 选择 latent thinking tokens（而非显式 text CoT）正是为了规避 reasoning collapse



### EvolveNav: Empowering LLM-Based Vision-Language Navigation via Self-Improving Embodied Reasoning

- **作者**: Bingqian Lin, Yunshuang Nie 等 | 通讯: **Xiaodan Liang**（中山大学 / MBZUAI）
- **发表**: arXiv 2506.01551
- **输入**: 全景 RGB + 导航指令
- **输出**: 推理链 + 导航动作
- **功能与效果**: 自改进推理范式。Stage 1 Formalized CoT SFT 激活推理能力；Stage 2 Self-Reflective Post-Training 通过对比正确/错误推理模式迭代改进。R2R/REVERIE/CVDN/SOON 上超越先前 LLM-based VLN
- **创新点**: (1) 将 self-reflection 引入 VLN 推理训练；(2) 无需外部 reward model 的自改进循环
- **与 NaviAgent 的关联**: SFT→迭代优化的训练范式参考



### Nav-R1: Reasoning and Navigation in Embodied Scenes

- **作者**: Qingxiang Liu, Ting Huang 等 | 通讯: **Hao Tang**（Peking University）
- **发表**: arXiv 2509.10884
- **输入**: Egocentric RGB 视频 + 目标描述
- **输出**: 推理文本 + 导航动作
- **功能与效果**: Fast-in-Slow 推理范式 + GRPO 强化学习。构建 Nav-CoT-110K 数据集。设计 format/understanding/navigation 三种 reward 分别约束推理格式、场景理解和导航结果。实机部署平均 +8%
- **创新点**: (1) Fast-in-Slow 双系统推理范式（语义推理慢、反应控制快）；(2) 结构化多 reward GRPO；(3) 大规模导航 CoT 数据集
- **与 NaviAgent 的关联**: 双系统思考设计与 NaviAgent 高度相关；GRPO + 多 reward 方案是 Stage 3 RFT 的参考



### VLN-R1: Vision-Language Navigation via Reinforcement Fine-Tuning

- **作者**: Zhangyang Qi, Zhixiong Zhang 等 | 通讯: **Hengshuang Zhao**（HKU / Shanghai AI Lab）
- **发表**: arXiv 2506.17221
- **输入**: Egocentric RGB 视频 + 导航指令
- **输出**: 连续导航动作序列
- **功能与效果**: 首个将 GRPO 应用于端到端 VLN。2B 模型通过 RFT 超越 7B SFT-only 模型。构建 VLN-Ego 数据集，提出 Long-Short Memory Sampling 平衡历史与当前观测
- **创新点**: (1) Time-Decayed Reward (TDR) 加权多步未来动作；(2) GRPO-based RFT 在 VLN 中的首次应用；(3) Long-Short Memory Sampling
- **与 NaviAgent 的关联**: 直接验证了 NaviAgent 的 SFT→RFT 训练管线。TDR 方案可用于 Stage 3 RFT



### ETP-R1: Evolving Topological Planning with Reinforcement Fine-tuning for VLN-CE

- **作者**: Shuhao Ye, Sitong Mao 等 | 通讯: **Yue Wang**（Zhejiang University）
- **发表**: arXiv 2512.20940
- **输入**: 拓扑图 + 全景 RGB + 导航指令
- **输出**: 图上的导航路径
- **功能与效果**: graph-based VLN-CE 上的 GRPO RFT。三阶段训练：Gemini API 大规模数据→联合预训练→闭环在线 RFT。R2R-CE 和 RxR-CE 全面 SOTA
- **创新点**: (1) 首次在 graph-based VLN-CE 上实现闭环在线 GRPO；(2) 用 Gemini API 自动生成大规模高质量预训练数据
- **与 NaviAgent 的关联**: 进一步验证 GRPO 在 VLN-CE 中的有效性



### JanusVLN: Decoupling Semantics and Spatiality with Dual Implicit Memory for VLN

- **作者**: Shuang Zeng, Dekang Qi 等 | 通讯: **Shuang Zeng**（Xi'an Jiaotong University）
- **发表**: ICLR 2026, arXiv 2509.22548
- **输入**: 仅 RGB 图像序列 + 导航指令（无需 depth）
- **输出**: 离散导航动作
- **功能与效果**: 双隐式记忆（空间-几何 KV cache + 视觉-语义 KV cache），仅保留 initial + sliding window token。VLN-CE SOTA，SR +3.6-35.5pp。受人脑左右半球功能特化启发
- **创新点**: (1) 双编码器分别处理 spatial-geometric 和 visual-semantic 信息；(2) KV cache 形式的隐式记忆（非显式图结构）；(3) 高效增量更新
- **与 NaviAgent 的关联**: 验证了隐式记忆（KV cache / learned embeddings）优于显式记忆（图结构）的假设



### MapNav: A Novel Memory Representation via Annotated Semantic Maps for VLM-based VLN

- **作者**: Lingfeng Zhang, Xiaoshuai Hao 等 | 通讯: **Renjing Xu**（HKUST(GZ) / Tsinghua）
- **发表**: ACL 2025, arXiv 2502.13451
- **输入**: RGB-D 图像序列 + 导航指令 → 在线构建 Annotated Semantic Map (ASM)
- **输出**: ASM 图像 + 当前 RGB → VLM 预测导航动作
- **功能与效果**: 用语义地图替代历史帧作为 VLN 记忆表征。仿真和真实环境 SOTA
- **创新点**: (1) 多通道 top-down 语义地图 + 文本标签标注；(2) 地图渲染为图像后通过标准 VLM 处理（无需特殊架构）
- **与 NaviAgent 的关联**: NaviAgent 的记忆工具设计参考了其语义地图概念，但用 learned embeddings 替代显式地图



### Mem4Nav: Boosting VLN in Urban Environments with Hierarchical Spatial-Cognition Long-Short Memory

- **作者**: Lixuan He, Haoyu Dong 等 | 通讯: **Yong Li**（Tsinghua University）
- **发表**: arXiv 2506.19433
- **输入**: 城市环境 RGB 图像 + 导航指令
- **输出**: 导航动作
- **功能与效果**: 可训练 memory tokens（通过 reversible Transformer 编码）用于城市 VLN。稀疏八叉树 + 语义拓扑图作为长期记忆，频率-新近度 cache 作为短期记忆。Touchdown/Map2Seq +7-13pp
- **创新点**: (1) Memory tokens 作为可训练嵌入注入 Transformer context（非外部数据结构）；(2) 可逆 Transformer 保证记忆无损编解码
- **与 NaviAgent 的关联**: "memory-as-context-tokens" 思路与 NaviAgent 一致



### RAGNav: A Retrieval-Augmented Topological Reasoning Framework for Multi-Goal VLN

- **作者**: Ling Luo, Qianqian Bai | 通讯: **Ling Luo**（西南财经大学）
- **发表**: arXiv 2603.03745
- **输入**: 多视角 RGB + 多目标导航指令 + 拓扑图 + 语义森林
- **输出**: 多目标导航的最优目标访问顺序 + 导航动作
- **功能与效果**: 双库记忆（拓扑图 + 语义森林）用于多目标 VLN。Anchor-guided retrieval + score propagation。Multi-goal VLN SOTA
- **创新点**: (1) 双基底记忆系统（物理连通性 + 语义层级抽象）；(2) 检索增强的多目标可达性推理
- **与 NaviAgent 的关联**: 按需检索记忆的思路与 `<tool:mem_r>` 一致



### TagaVLM: Topology-Aware Global Action Reasoning for Vision-Language Navigation

- **作者**: Jiaxing Liu, Zexi Zhang 等 | 通讯: **Xiaoyan Li**（北京工业大学）
- **发表**: arXiv 2603.02972
- **输入**: 全景 RGB + 导航指令 + 拓扑图结构
- **输出**: 图上的导航方向选择
- **功能与效果**: 将拓扑结构显式注入 VLM backbone。R2R SOTA（大模型方法中）
- **创新点**: (1) 拓扑结构 token 化注入 VLM；(2) 结构化空间推理
- **与 NaviAgent 的关联**: 验证了拓扑信息对 VLM 导航决策的增益



### Uni-NaVid: A Video-based Vision-Language-Action Model for Unifying Embodied Navigation Tasks

- **作者**: Jiazhao Zhang, Kunyu Wang 等 | 通讯: **He Wang**（Peking University, PKU-EPIC）
- **发表**: RSS 2025
- **输入**: RGB 视频流 + 自然语言目标/指令
- **输出**: 离散导航动作（文本 token 形式）
- **功能与效果**: 统一 VLN + ObjectNav + EQA + Human Following 的视频基导航模型。3.6M 样本，5Hz。ObjectNav 数据用 L3MVN 启发式专家生成
- **创新点**: (1) Online token merge 策略压缩历史视频帧；(2) 四种导航任务统一为 video-in, text-out 格式
- **与 NaviAgent 的关联**: ObjectNav 数据生成（L3MVN）和统一多任务训练的参考



### NavFoM: Embodied Navigation Foundation Model

- **作者**: Jiazhao Zhang, Anqi Li 等 | 通讯: **He Wang**（Peking University, PKU-EPIC）
- **发表**: arXiv 2509.12129
- **输入**: 多视角 RGB 视频 + 目标描述/指令 + TVI tokens
- **输出**: 8-step waypoint trajectory (x, y, theta)，通过 planning head (MLP) 从 action hidden state 回归
- **功能与效果**: 跨形态（四足/无人机/轮式/车辆）、跨任务导航基础模型。8M 样本训练。7 个 benchmark SOTA 或极具竞争力，无需任务微调
- **创新点**: (1) Temporal-Viewpoint Indicator (TVI) tokens——可学习嵌入编码时间和视角；(2) Budget-Aware Temporal Sampling (BATS) 动态控制 token 预算；(3) 统一 waypoint trajectory 输出格式
- **与 NaviAgent 的关联**: NaVocab 的 context token 设计参考 TVI。NavFoM 用 L3MVN 生成 ObjectNav 训练轨迹的方案也是 NaviAgent 室内探索数据的参考

## 3. 室内导航：无图探索式（ObjectNav / Goal-Nav）

> 给定目标描述，无路线指令，自主探索环境。含 Frontier+VLM、VLM 微调、端到端 VLA、认知/层次化系统。


### CogNav: Cognitive Process Modeling for Object Goal Navigation with LLMs

- **作者**: Yihan Cao, Jiazhao Zhang 等 | 通讯: **Kai Xu**（国防科技大学 NUDT）
- **发表**: ICCV 2025, arXiv 2412.10439
- **输入**: Posed RGB-D 帧（每步）→ 在线构建异质认知地图（场景图 + 地标图 + 占据地图）→ 文本化 prompt 送入 GPT-4V/GPT-3
- **输出**: 认知状态转移决策（Broad Search / Contextual Search / Observe Target / Verification / Confirmation）+ 选择的导航地标坐标 → 局部路径规划器执行
- **功能与效果**: 零样本 ObjectNav。HM3D SR 72.5%（+14% relative）。**不做任何训练/微调**，完全基于 prompting

### BridgeNav: Bridging the Indoor-Outdoor Gap — Vision-Centric Instruction-Guided Embodied Navigation for the Last Meters

- **作者**: Yuxiang Zhao*, Yirong Yang*（共一）等 | 通讯: **Mu Xu**（AMAP CV Lab, Alibaba Group）
- **发表**: arXiv 2602.06427
- **任务类型**: **Out-to-In Prior-Free Goal-Conditioned Navigation**——Agent 在目标建筑附近（室外），仅凭第一人称视觉和简单目标指令（如 "go to Starbucks"），**无 GPS、无地图、无精确坐标**，自主找到并进入目标建筑入口。这是配送机器人"最后几十米"的关键问题
- **输入**: Egocentric RGB 视频帧序列 + 自然语言目标指令。初始观测 + 历史帧（h=10 帧 context）
- **输出**: 未来 5 步的 waypoint 序列 (x, y, z, yaw)，在 Euclidean 空间中表示
- **功能与效果**: 基于 Qwen2.5-VL-3B。发布首个 out-to-in 数据集 BridgeNavDataset（55K 轨迹，10M+ 帧）。数据通过 trajectory-conditioned video synthesis（Wan2.1-I2V Diffusion Transformer）生成，而非真实采集。定量超越 SOTA baselines
- **创新点**: (1) **Latent Intention Inference**：根据导航阶段动态关注图像不同区域——远处判断目标是否在视野中→中距看招牌辨别方向→近处找入口精确进入。这种"注意力随距离动态演化"的设计很有启发性；(2) **光流引导动态感知**：用 RAFT 计算光流，预测哪些图像区域会因自身运动发生最大视觉变化，让 agent 学会"想象"运动后的视觉变化；(3) 首个 out-to-in 开源数据集，创新性地使用 video diffusion 生成数据
- **与 NaviAgent 的关联**: 最接近 NaviAgent 的**探索式导航**理念——都是 prior-free、instruction-driven、无需 turn-by-turn 指令。BridgeNav 的 Latent Intention（动态注意力）与 NaviAgent 的 latent thinking tokens（隐式推理）在概念上呼应。**关键区别**：BridgeNav 仅处理 out-to-in 最后几十米过渡（假设 agent 已在目标建筑附近），不涉及：(1) 室内→室外方向；(2) 长程导航（200-2000m）；(3) Agent 推理和工具调用；(4) 空间记忆和回环检测。NaviAgent 处理完整的双向跨环境链路



### VAMOS: A Hierarchical Vision-Language-Action Model for Capability-Modulated and Steerable Navigation

- **作者**: Mateo Guaman Castro 等 | 通讯: **Abhishek Gupta**（University of Washington）
- **发表**: arXiv 2510.20818

- **功能与效果**: 跨室内外 + 跨腿式/轮式机器人的统一导航。VLM pixel path + affordance model
- **创新点**: (1) 高层语义路径 + 低层 affordance 调制的两级架构；(2) 跨 embodiment 的 affordance 适配
- **与 NaviAgent 的关联**: 少数跨室内外工作之一，验证了统一跨环境导航的可行性

### ReAct: Synergizing Reasoning and Acting in Language Models

### ARNA: General-Purpose Robotic Navigation via LVLM-Orchestrated Perception, Reasoning, and Acting

- **作者**: Bernard Lange*, Anil Yildiz, Mansur Arief 等 | 通讯: **Bernard Lange**（Stanford / JPL-Caltech）

### CoINS: Counterfactual Interactive Navigation via Skill-Aware VLM

- **作者**: Kangjie Zhou, Zhejia Wen 等 | 通讯: **Chang Liu**（Peking University）
- **发表**: arXiv 2601.03956
- **输入**: RGB 图像 + 交互式导航指令
- **输出**: 技能选择 + 执行参数
- **功能与效果**: Skill-aware VLM + RL 技能库，VLM 通过反事实推理决定何时调用哪个技能（如开门、按电梯等）。适用于需要物理交互的导航场景
- **创新点**: (1) 反事实交互推理（counterfactual reasoning）用于技能选择；(2) VLM 与 RL 技能库的层级协作
- **与 NaviAgent 的关联**: "VLM 决策 + 技能执行"范式与 NaviAgent 的 tool token 思路相近



### ToolkenGPT: Augmenting Frozen Language Models with Massive Tools via Tool Embeddings

- **作者**: Shibo Hao, Tianyang Liu 等 | 通讯: **Zhiting Hu**（UC San Diego）
- **发表**: NeurIPS 2023 (Oral)
- **输入**: 文本 prompt（与标准 LLM 一致）
- **输出**: 文本 token + tool token（自动交替）。当模型预测出 toolken 时，切换到参数生成模式，调用外部工具后将结果注入继续生成
- **功能与效果**: 在 frozen LLM 上通过仅训练 toolken embeddings 实现大规模工具调用（可扩展到数千工具）。GSM8K、FuncQA、VirtualHome 等基准上超越 ReAct
- **创新点**: (1) Tool-as-token 范式：每个工具是词表中的一个 learned embedding；(2) 仅训练 toolken embeddings（LLM 冻结），极其轻量
- **与 NaviAgent 的关联**: NaviAgent 的 NaVocab tool token 范式直接源于此。区别：NaviAgent 做 full SFT（非 frozen LLM）+ 导航专用可微工具



### ToolGen: Unified Tool Retrieval and Calling via Generation

- **创新点**: (1) 三阶段训练（工具记忆化→检索训练→Agent 训练）；(2) 大规模工具 token 化（47K 工具）
- **与 NaviAgent 的关联**: 验证了 tool token 方案的可扩展性



### Re-Initialization Token Learning for Tool-Augmented Large Language Models

- **作者**: Chenghao Li, Liu Liu 等 | 通讯: **Liu Liu**（北京航空航天大学）

## 6. VLA 架构设计：Tokenization / Reasoning / Memory / Depth

> 动作 tokenization、latent reasoning、可微记忆、Depth 注入等架构组件，主要在操作任务上验证。


### FAST: Efficient Action Tokenization for Vision-Language-Action Models

- **作者**: Karl Pertsch, Kyle Stachowicz 等 | 通讯: **Karl Pertsch**（Physical Intelligence / UC Berkeley）
- **发表**: arXiv 2501.09747
- **输入**: 连续动作轨迹（机器人关节序列）
- **输出**: 压缩后的动作 token 序列（30-60 tokens per action chunk，vs 原始 256+ tokens）
- **功能与效果**: DCT + BPE 动作 tokenization。10x 压缩率，5x 训练加速，15x 推理加速。FAST+ 通用 tokenizer 在 1M 真实轨迹上训练，可作为 black-box 复用

- **与 NaviAgent 的关联**: 动作编码参考。NaviAgent 的 pixel goal 较简单（2D 坐标），不需要 FAST 级别的压缩

### SpatialVLA: Exploring Spatial Representations for Visual-Language-Action Model

- **作者**: Delin Qu, Haoming Song 等 | 通讯: **Xuelong Li**（Shanghai AI Lab / 西北工大）
